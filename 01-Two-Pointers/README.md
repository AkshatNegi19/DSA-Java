# Two Pointers

## 📖 Overview

The Two Pointer pattern is used to solve problems by maintaining two indices that move through an array or string. It helps reduce brute-force O(n²) solutions to O(n).

---

## When to Use

- Sorted arrays
- Finding pairs
- Removing duplicates
- Palindrome problems
- Maximizing or minimizing values

---

## Common Problems

- Two Sum II
- Container With Most Water
- Valid Palindrome
- Remove Duplicates from Sorted Array
- Squares of a Sorted Array

---

## Time Complexity

Generally O(n)

---

## Space Complexity

Usually O(1)
