# Graphs

## 📖 Overview

Graphs model relationships between nodes and edges. They are commonly solved using BFS, DFS, Topological Sort, Union-Find, and Shortest Path algorithms.

---

## Topics Covered

- BFS
- DFS
- Topological Sort
- Union Find
- Cycle Detection
- Shortest Path

---

## Common Problems

- Number of Islands
- Clone Graph
- Course Schedule
- Number of Provinces
- Rotting Oranges
- Word Ladder

---

## Time Complexity

Depends on the algorithm.

Most BFS and DFS traversals run in **O(V + E)**.
