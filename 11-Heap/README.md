# Heap / Priority Queue

## 📖 Overview

Heap efficiently maintains the smallest or largest element.

---

## When to Use

- Top K Problems
- Scheduling
- Merge Sorted Lists
- Median Problems

---

## Common Problems

- Kth Largest Element
- Top K Frequent Elements
- Merge K Sorted Lists
- Find Median from Data Stream

---

## Time Complexity

Insertion and Removal: O(log n)
