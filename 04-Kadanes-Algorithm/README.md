# Kadane's Algorithm

## 📖 Overview

Kadane's Algorithm finds the maximum sum contiguous subarray in linear time.

---

## When to Use

- Maximum Subarray Sum
- Circular Maximum Sum
- Dynamic Programming optimization

---

## Common Problems

- Maximum Subarray
- Maximum Circular Subarray

---

## Time Complexity

O(n)

---

## Space Complexity

O(1)
