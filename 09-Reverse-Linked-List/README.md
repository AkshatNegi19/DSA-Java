# In-place Reversal of Linked List

## 📖 Overview

Uses pointer manipulation to reverse a linked list without extra memory.

---

## When to Use

- Reverse Linked List
- Reverse Between
- Reverse K Group

---

## Common Problems

- Reverse Linked List
- Reverse Linked List II
- Reverse Nodes in K Group

---

## Time Complexity

O(n)

---

## Space Complexity

O(1)
