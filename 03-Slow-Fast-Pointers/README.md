# Slow & Fast Pointers

## 📖 Overview

Uses two pointers moving at different speeds, mainly in linked lists or sequences.

---

## When to Use

- Cycle Detection
- Finding Middle Node
- Happy Number
- Detecting Loops

---

## Common Problems

- Linked List Cycle
- Happy Number
- Middle of Linked List
- Circular Array Loop

---

## Time Complexity

O(n)

---

## Space Complexity

O(1)
