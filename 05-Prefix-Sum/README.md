# Prefix Sum

## 📖 Overview

Prefix Sum stores cumulative sums allowing range queries to be answered efficiently.

---

## When to Use

- Range Sum Queries
- Subarray Sum
- Frequency Counting

---

## Common Problems

- Range Sum Query
- Subarray Sum Equals K
- Continuous Subarray Sum

---

## Time Complexity

O(n)

---

## Space Complexity

O(n)
