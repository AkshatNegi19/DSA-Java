# Binary Search

## 📖 Overview

Binary Search repeatedly divides the search space in half.

---

## When to Use

- Sorted Arrays
- Search Space Problems
- Answer Binary Search

---

## Common Problems

- Binary Search
- Search Insert Position
- First Bad Version
- Koko Eating Bananas

---

## Time Complexity

O(log n)

---

## Space Complexity

O(1)
