# Sliding Window

## 📖 Overview

Sliding Window maintains a window over contiguous elements and expands or shrinks it depending on the condition.

---

## When to Use

- Subarrays
- Substrings
- Fixed-size windows
- Variable-size windows

---

## Common Problems

- Maximum Average Subarray
- Longest Substring Without Repeating Characters
- Minimum Size Subarray Sum
- Permutation in String

---

## Time Complexity

Usually O(n)

---

## Space Complexity

O(1) to O(k)
